# Elevia Admin App

Application mobile d'administration interne pour **Elevia Agency**.  
Développée en Flutter dans le cadre d'un cours de développement mobile.

## Fonctionnalités prévues

- **Dashboard** — chiffres clés (CA, devis en cours, tâches)
- **CRM** — gestion des clients et contacts
- **Facturation** — suivi des devis et factures
- **Tâches** — gestion des projets et tâches

## Stack technique

- Flutter (Dart)
- Navigation : go_router
- State management : Provider

## Lancer le projet

```bash
flutter pub get
flutter run
```
